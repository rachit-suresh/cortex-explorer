# Cortex Explorer

> Interactive Interest Graph Explorer with a Neo-Brutalist aesthetic.

**Cortex** is a visual engine for exploring personal interest topologies. It replaces standard list-based UI with an interactive, directed graph. Built on the VITE stack, it utilizes a custom recursive layout algorithm to map relationships between domains (e.g., Sports -> Racing -> F1).
